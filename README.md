# Tango Contemporáneo – Rhythmus-Trainer

Ein Rhythmus-Trainer für Tango Contemporáneo mit Paar-Ansicht, Schrittmustern
(Wiegeschritt, Caminada, Marcha, Synkope, Ocho atrás, Ocho adelante) und
synthetischem Bandoneón-Sound über die Web Audio API.

Die App ist eine einzelne statische `index.html` — keine Abhängigkeiten,
kein Build-Schritt.

## Über GitHub Pages veröffentlichen

1. Im Repository auf **Settings → Pages** gehen.
2. Unter **Build and deployment** als Source **Deploy from a branch** wählen.
3. Als Branch den gewünschten Branch (z. B. `main`) und den Ordner **/ (root)** auswählen, dann **Save**.
4. Nach kurzer Zeit ist die App unter `https://<benutzername>.github.io/<repository>/` erreichbar.

## Lokal ausprobieren

Einfach `index.html` im Browser öffnen — mehr ist nicht nötig.
